You will get an odd integer `n` (>= 3) and your task is to draw an X. Each line is separated with `\n`.

Use the following characters: <code>■ □</code> 
For Ruby, Crystal and PHP:  <code>whitespace and o</code>

## Examples

<pre>
<code>
                                     ■□□□■
            ■□■                      □■□■□
  x(3) =>   □■□             x(5) =>  □□■□□
            ■□■                      □■□■□
                                     ■□□□■
</code>
