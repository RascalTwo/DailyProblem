Given an array `arr` of unique nonnegative integers, implement a function that finds the smallest nonnegative integer that is NOT in the array.

## Examples

| Input                   | Output |
| ----------------------- | ------ |
| `[0, 1, 2, 3]`          | `4`    |
| `[0]`                   | `1`    |
| `[0, 1, 2]`             | `3`    |
| `[1, 3, 0, 2]`          | `4`    |
| `[1000000]`             | `0`    |
| `[1, 0, 3, 4, 5]`       | `2`    |
| `[0, 1000000]`          | `1`    |
| `[0, 99999, 1000000]`   | `1`    |
| `[0, 5, 4, 1, 3, 6, 2]` | `7`    |
