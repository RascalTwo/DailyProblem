## Examples

| Input                                                | Output |
| ---------------------------------------------------- | ------ |
| `Node(1, Node(2, Node(3)), Node(4, None, Node(5)))`  | `15`   |
| `Node(1, Node(2, Node(3)), Node(4, None, Node(-5)))` | `5`    |
