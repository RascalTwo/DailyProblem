Given an string of characters, ensure no consecutive duplicate characters existing by removing both offending characters.

## Examples

| Input      | Output  |
| ---------- | ------- |
| `'abbc'`   | `'ac'`  |
| `'abccbc'` | `'ac'`  |
| `'abbbbc'` | `'ac'`  |
| `'abbbc'`  | `'abc'` |
