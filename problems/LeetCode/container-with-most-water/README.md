Given a list of integers where values represent heights, find the two heights which together form a container that con contain the most liquid.

## Examples

| Input                         | Output   |
| ----------------------------- | -------- |
| `[1, 8, 6, 2, 5, 4, 8, 3, 7]` | `(1, 8)` |
| `[1, 1]`                      | `(0, 1)` |
| `[4, 3, 2, 1, 4]`             | `(0, 4)` |
| `[1, 2, 1]`                   | `(0, 2)` |
