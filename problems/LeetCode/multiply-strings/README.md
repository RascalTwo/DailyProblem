Given two non-negative represented as strings, return their product, also represented as a string.

## Examples

| Input                                    | Output      |
| ---------------------------------------- | ----------- |
| `['2', '3']`                             | `'6'`       |
| `['123', '456']`                         | `'56088'`   |
| `['1', '2', '3', '4', '5', '67', '890']` | `'7155600'` |
| `['0', '0']`                             | `'0'`       |
