Given a string of round, curly, and square open and closing parentheses, return whether the parentheses are balanced (well-formed).

## Examples

| Input          | Output  |
| -------------- | ------- |
| `'([])[]({})'` | `True`  |
| `'([)]'`       | `False` |
| `'((()'`       | `False` |
| `')'`          | `False` |
