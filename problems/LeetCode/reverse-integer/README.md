Given a signed 32-bit integer, return it with its digits reversed.

If reversing causes the value to go outside the signed 32-bit integer range, then return 0.

## Examples

| Input        | Output |
| ------------ | ------ |
| `123`        | `321`  |
| `-123`       | `-321` |
| `120`        | `21`   |
| `2147483647` | `0`    |
