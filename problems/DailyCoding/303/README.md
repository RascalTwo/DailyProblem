Given a clock time in `hh:mm` format, determine, to the nearest degree, the angle between the hour and the minute hands.

> Bonus: When, during the course of a day, will the angle be zero?

## Examples

| Input     | Output |
| --------- | ------ |
| `'12:00'` | `0`    |
| `'6:00'`  | `180`  |
| `'6:30'`  | `15`   |
| `'9:45'`  | `22.5` |
