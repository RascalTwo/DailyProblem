The power set of a set is the set of all its subsets.

Write a function that, given a set, generates its power set.

## Examples

| Input       | Output                                                   |
| ----------- | -------------------------------------------------------- |
| `{1, 2, 3}` | `[{}, {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3}]` |
