Compute the running median of a sequence of numbers.

That is, given a stream of numbers, print out the median of the list so far on each new element.

## Examples

| Input                   | Output                      |
| ----------------------- | --------------------------- |
| `[2, 1, 5, 7, 2, 0, 5]` | `[2, 1.5, 2, 3.5, 2, 2, 2]` |
