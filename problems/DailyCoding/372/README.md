Write a function that takes a natural number as input and returns the number of digits the input has.

## Examples

| Input | Output |
| ----- | ------ |
| `1`   | `1`    |
| `9`   | `1`    |
| `10`  | `2`    |
| `0`   | `1`    |
| `-10` | `2`    |
| `-99` | `2`    |
| `100` | `3`    |
| `150` | `3`    |
