Given a string with repeated characters, rearrange the string so that no two adjacent characters are the same.

If this is not possible, return `None`.

## Examples

| Input      | Output     |
| ---------- | ---------- |
| `'aaabbc'` | `'ababac'` |
| `'aaab'`   | `None`     |
