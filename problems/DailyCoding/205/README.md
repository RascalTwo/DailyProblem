Given an integer, find the next permutation of it in absolute order.

## Examples

Given `48975`, the next permutation would be `49578`.

| Input   | Output  |
| ------- | ------- |
| `48975` | `49578` |
| `123`   | `132`   |
| `978`   | `987`   |
| `654`   | `654`   |
