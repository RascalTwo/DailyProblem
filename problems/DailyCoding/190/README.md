Given a circular array, compute its maximum subarray sum.

A subarray can be empty, and in this case the sum is `0`.

## Examples

| Input           | Output |
| --------------- | ------ |
| `[8, -1, 3, 4]` | `15`   |
| `[-4, 5, 1, 0]` | `6`    |
