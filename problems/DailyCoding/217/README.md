We say a number is sparse if there are no adjacent ones in its binary representation.

For a given input `N`, find the smallest sparse number greater than or equal to `N`.

## Examples

| Input | Output |
| ----- | ------ |
| `21`  | `21`   |
| `22`  | `32`   |
