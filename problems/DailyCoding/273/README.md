A fixed point in an array is an element whose value is equal to its index.

Given a sorted array of distinct elements, return a fixed point, if one exists.

Otherwise, return `-1`.

## Examples

| Input                | Output |
| -------------------- | ------ |
| `[7, 8, 5, 2, 4, 9]` | `4`    |
| `[7, 8, 5, 2, 8, 9]` | `-1`   |
| `[-6, 0, 2, 40]`     | `2`    |
| `[1, 5, 7, 8]`       | `-1`   |
