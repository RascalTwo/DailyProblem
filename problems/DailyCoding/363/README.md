Write a function, `add_subtract`, which alternately adds and subtracts curried arguments.

## Examples

| Input                        | Output |
| ---------------------------- | ------ |
| `add_subtract(7)`            | `7`    |
| `add_subtract(1)(2)(3)`      | `0`    |
| `add_subtract(-5)(10)(3)(9)` | `11`   |
