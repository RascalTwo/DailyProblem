Write an algorithm that finds the total number of set bits in all integers between `1` and `N`.

## Examples

| Input     | Output    |
| --------- | --------- |
| `2`       | `1`       |
| `10`      | `15`      |
| `16`      | `32`      |
| `100`     | `316`     |
| `1000`    | `4932`    |
| `1024`    | `5120`    |
| `1000000` | `9884992` |
