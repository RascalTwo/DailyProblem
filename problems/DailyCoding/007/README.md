Given the mapping `a = 1`, `b = 2`, ... `z = 26`, and an encoded message, count the number of ways it can be decoded.

## Examples

The message `'111'` would give `3`, since it could be decoded as `'aaa'`, `'ka'`, and `'ak'`.

| Input    | Output |
| -------- | ------ |
| `'1'`    | `1`    |
| `'12'`   | `3`    |
| `'111'`  | `3`    |
| `'1111'` | `5`    |
| `'121'`  | `8`    |
