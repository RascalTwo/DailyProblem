Given a list of strictly positive integers, partition the list into 3 contiguous partitions which each sum up to the same value. If not possible, return `None`.

## Examples

| Input             | Output                  |
| ----------------- | ----------------------- |
| `[3, 5, 8, 0, 8]` | `[[3, 5], [8, 0], [8]]` |
