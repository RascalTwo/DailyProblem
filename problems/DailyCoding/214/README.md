Given an integer `n`, return the length of the longest consecutive run of `1`s in its binary representation.

## Examples

| Input | Output |
| ----- | ------ |
| `156` | `3`    |
| `1`   | `1`    |
| `3`   | `2`    |
| `7`   | `3`    |
| `15`  | `4`    |
| `31`  | `5`    |
| `63`  | `6`    |
