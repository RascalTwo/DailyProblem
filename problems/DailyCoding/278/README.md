Given an integer `N`, construct all possible binary search trees with `N` nodes.

## Examples

| Input | Output                                                                                                     |
| ----- | ---------------------------------------------------------------------------------------------------------- |
| `1`   | `[Node()]`                                                                                                 |
| `2`   | `[Node(Node()), Node(None, Node())]`                                                                       |
| `3`   | `[Node(Node(Node())), Node(None, Node(Node())), Node(Node(None, Node())), Node(None, Node(None, Node()))]` |
