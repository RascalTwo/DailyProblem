Given an array of integers, find the maximum XOR of any two elements.

## Examples

| Input             | Output |
| ----------------- | ------ |
| `[2, 2]`          | `0`    |
| `[2, 4]`          | `6`    |
| `[1, 2, 3, 4, 5]` | `7`    |
