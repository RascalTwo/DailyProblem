Given two singly linked lists that intersect at some point, find the intersecting node.

> The lists are non-cyclical.

## Examples

| Input                                                                       | Output |
| --------------------------------------------------------------------------- | ------ |
| `Node(3, Node(7, Node(8, Node(10)))), Node(99, Node(1, Node(8, Node(10))))` | `8`    |
