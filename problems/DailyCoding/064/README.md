A knight's tour is a sequence of moves by a knight on a chessboard such that all squares are visited once.

Given `N`, write a function to return the number of knight's tours on an `N` by `N` chessboard.

## Examples

| Input | Output |
| ----- | ------ |
| `1`   | `1`    |
| `2`   | `0`    |
| `3`   | `10`   |
| `4`   | `16`   |
