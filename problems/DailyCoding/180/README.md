Given a stack of `N` elements, interleave the first half of the stack with the second half reversed using only one other queue.

## Examples

| Input             | Output            |
| ----------------- | ----------------- |
| `[1, 2, 3, 4, 5]` | `[1, 5, 2, 4, 3]` |
| `[1, 2, 3, 4]`    | `[1, 4, 2, 3]`    |
