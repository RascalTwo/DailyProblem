Given a string, split it into as few strings as possible such that each string is a palindrome.

## Examples

| Input                | Output                         |
| -------------------- | ------------------------------ |
| `'racecarannakayak'` | `['racecar', 'anna', 'kayak']` |
| `'abc'`              | `['a', 'b', 'c']`              |
