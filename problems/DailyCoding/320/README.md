Given a string, find the smallest window that contains every distinct character.

Characters may appear more than once in the window.

## Examples

| Input        | Output               |
| ------------ | -------------------- |
| `"jiujitsu"` | `(2, 6)` or `(3, 7)` |
| `"racecar"`  | `(0, 3)` or `(3, 6)` |
