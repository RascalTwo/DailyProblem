Given an array of strictly the characters `'R'`, `'G'`, and `'B'`, segregate the values of the array so that all the `R`s come first, the `G`s come second, and the `B`s come last.

## Examples

| Input                                 | Output                                |
| ------------------------------------- | ------------------------------------- |
| `['G', 'B', 'R', 'R', 'B', 'R', 'G']` | `['R', 'R', 'R', 'G', 'G', 'B', 'B']` |
